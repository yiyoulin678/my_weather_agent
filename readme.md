# MCP Weather Agent

## Introduction

MCP Weather Agent is a travel assistant based on the Model Context Protocol (MCP). It can provide real-time temperature information and weather risk analysis for users.

## Features

- Get current temperature of a city
- Analyze weather risks for outdoor activities

## Tools

### get_temperature(city)
Returns the current temperature.

### get_weather_alert(city)
Analyzes the weather condition and determines possible risks.

## Technology Stack

- Python
- MCP SDK
- Pydantic
- wttr.in API
